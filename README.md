# Recession

This repo contains models for probability of recession and time until recession in jupyter notebooks using pandas, numpy, sklearn, and tensorflow in python. Various classical ML models are used and compared such as logistic, random forest, SVC, voting classifier, and boosting. Then a basic multi-layer perceptron neural network is modeled and compared as well.

\data\ contains data input file.

\src\ contains the jupyter notebooks for the models. "Recession" notebooks are for probability of recession models and "RecTime" are for time until recession models.
